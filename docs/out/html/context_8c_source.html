2 * This manages the symbolic context as a double linked list. The actual
3 * forward or "next" pointer is located in the AST. The "prev" pointer is
4 * located in the context data structure. When a symbol is looked up, it is
5 * only searched from the current context toward the root, so the prev pointer
6 * if the one that matters for this data structure.
7 */
